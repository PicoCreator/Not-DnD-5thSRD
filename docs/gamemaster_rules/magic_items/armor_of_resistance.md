# Armor of Defiance
_Armor (light, medium, or heavy), rare (requires attunement)_

Whilst wearing this armor, you have resistance to one type of damage. The GM shall choose the type or randomly select from the options below.

| d10 | Damage Type |
|-----|-------------|
| 1   | Acid        |
| 2   | Cold        |
| 3   | Fire        |
| 4   | Force       |
| 5   | Lightning   |
| 6   | Necrotic    |
| 7   | Poison      |
| 8   | Psychic     |
| 9   | Radiant     |
| 10  | Thunder     |
